Currently, the C# SDK runs on macOS, Windows, and Linux. There is no peer to peer capabilites of the C# SDK, however we are working on it. See below for [Xamarin support](#xamarin).

The C# SDK is available as part of [NuGet](https://nuget.org). The common ways to install the SDK is with either the NuGet Package Manager, the .NET CLI, or by adding a reference to the CSProj XML file.

## Xamarin

:::info
The device simulator for Xamarin-built applications will not work with Ditto. You have to run your Xamarin application on a physical device.
:::

Follow the [Xamarin tutorial](./tutorial/xamarin) to build your first Tasks application with Ditto.
